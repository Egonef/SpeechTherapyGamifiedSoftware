# TFG - Gamified Speech Therapy (scaffold)

Estructura inicial del proyecto. Contiene dos secciones principales: `games/` y `analysis/`.

Cómo usar (desarrollo local):

1. Abrir `index.html` en un navegador (por ejemplo con Live Server o abrir el fichero directamente).
2. Navegar a `src/games/index.html` para trabajar en los juegos.
3. Navegar a `src/analysis/index.html` para trabajar en el dashboard.

Se ha incluido un ejemplo mínimo de `game1` y datos de ejemplo en `data/sample-results.json`.
# SpeechTherapyGamifiedSoftware
A gamified software system on speech therapy aiming for the phonological, syntactic, and cognitive development of children, including performance analysis and graphical data visualization.
